# HELPER Web Reliability 50 Public Proof Bundle

This folder is a publication-ready public proof bundle for HELPER research/web reliability.

It intentionally contains only public-safe exports:

- frozen corpus and target lock
- sanitized full-50 results
- summary note
- manifest and checksums

It intentionally excludes raw private-core runtime payloads:

- request envelopes
- raw response DTOs
- conversation and turn identifiers
- runtime logs and audit traces
- absolute local machine paths
